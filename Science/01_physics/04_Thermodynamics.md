## 🌟 Thermodynamic Laws - Quick Summary

### 🥇 **Zeroth Law** 
```
    A ≡ B, B ≡ C → A ≡ C
    (Thermal equilibrium transitivity)
```

### 🥈 **First Law** 
```
    ΔU = Q - W
    (Energy conservation - pehla niyam)
```

### 🥉 **Second Law**
```
    ΔS_universe ≥ 0
    (Entropy always increases - doosra niyam)
```

### 🏆 **Third Law**
```
    S(T→0) = 0
    (Perfect crystal entropy at absolute zero)
```
